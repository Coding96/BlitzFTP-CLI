What We Hope To Achieve

A common problem with file sharing over the internet is server cost and legality
A full mesh network that requires no server after an incubation period is the end goal
This allows large files to be sent from 1 user to another without ever contacting a server
Ideally after some initial help from a server all information neccessary to finding a peer will be availble from other peers in the network
This is a large project that I expect will change over its development but please do get involved and please do add to the project as you see fit This project will be written in C++ using QT for a graphical interface once the CLI project is done
Have fun and don't work too hard
